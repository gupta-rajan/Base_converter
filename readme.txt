Digital System's Project (Base Converter)
_____________________________________________________________

Description : It is an interface for converting a number (either integer or float) from one base to another.
The project contains 3 files index.html, script.js and style.css with the following roles:

1. index.html and style.css are used for the styling and structure of the website
2. script.js contains the code for base conversion.

How to run the project?

1. Extract the downloaded folder 210020041.
2. Open any code editor (e.g VS code) then open this folder(210020041) in it.
3. Click on GoLive.

An interface for base conversion is now visible on your screen.
